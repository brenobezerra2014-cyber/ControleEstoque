# Controle de Estoque com Arquivos Binários

## Integrantes

BRENO DE ARAUJO BEZERRA

## Descrição

Sistema de controle de estoque desenvolvido em linguagem C utilizando arquivos binários.

O sistema possui dois tipos de registros:

* Produto
* Venda

Cada venda armazena um vetor contendo os IDs dos produtos vendidos.

Funcionalidades:

* Cadastro de produtos
* Consulta de produtos
* Atualização de produtos
* Remoção de produtos
* Cadastro de vendas
* Consulta de vendas
* Atualização de vendas
* Remoção de vendas
* Relatório relacionando vendas e produtos

Disciplina: C
